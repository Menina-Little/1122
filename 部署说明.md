# 🚀 网站部署说明

## 📁 网站文件说明

你现在拥有一个完整的静态网站，包含以下文件：

- **index.html** - 网站主页（这是你要找的文件！）
- **style.css** - 所有样式和动画效果
- **script.js** - 游戏逻辑和交互功能

## 💻 本地查看网站

### 方法1: 直接打开（最简单）
1. 双击打开 `index.html` 文件
2. 网站会在你的默认浏览器中打开
3. 点击"开始游戏"按钮即可玩贪吃蛇！

### 方法2: 使用本地服务器（推荐）
如果你有 Python：
```bash
# 在 E:\0menina 目录下打开命令行，运行：
python -m http.server 8000
```
然后访问：http://localhost:8000

## 🌐 部署到Vercel

### 方式1: 通过网页上传（最简单）

1. 访问 https://vercel.com 并登录（可以用GitHub账号）

2. 点击 "Add New..." → "Project"

3. 选择 "Deploy from a public Git repository" 或者直接拖拽文件

4. 如果选择上传文件：
   - 将以下文件一起上传：
     - index.html
     - style.css
     - script.js

5. Vercel会自动部署，几秒钟后你就有一个在线网站了！

### 方式2: 通过GitHub（推荐）

1. **创建GitHub仓库**
   - 访问 https://github.com
   - 点击 "New repository"
   - 给仓库起个名字，比如 "ai-portfolio"
   - 点击 "Create repository"

2. **上传代码到GitHub**
   在 E:\0menina 文件夹中打开命令行（或Git Bash），运行：
   ```bash
   git init
   git add index.html style.css script.js 部署说明.md
   git commit -m "初次提交：古风AI作品集网站"
   git branch -M main
   git remote add origin https://github.com/你的用户名/ai-portfolio.git
   git push -u origin main
   ```

3. **在Vercel中导入**
   - 访问 https://vercel.com
   - 点击 "Import Project"
   - 选择你的GitHub仓库
   - 点击 "Deploy"
   - 完成！你的网站现在在线了

### 方式3: 使用Vercel CLI

```bash
# 安装Vercel CLI
npm install -g vercel

# 在项目目录下运行
vercel

# 按照提示操作即可
```

## 🎨 网站功能

✅ **已实现的功能：**
- Vanta.js 飞鸟动画背景
- 霓虹渐变标题和故障特效
- 游戏卡片悬停发光效果
- 完整的贪吃蛇游戏（支持键盘和触摸控制）
- 完美的响应式设计（手机、平板、电脑都能用）

## 📱 如何在手机上查看

1. **本地测试**：
   - 确保手机和电脑在同一WiFi
   - 在电脑上运行本地服务器（见上面方法2）
   - 在手机浏览器输入：`http://你的电脑IP:8000`

2. **在线访问**：
   - 部署到Vercel后，你会得到一个链接（例如：your-site.vercel.app）
   - 在手机浏览器中直接访问这个链接即可

## 🎮 游戏操作说明

- **电脑**：方向键控制，空格键暂停
- **手机**：在游戏画布上滑动控制方向

## ⚠️ 注意事项

1. 需要联网才能加载 Vanta.js 和 Three.js 库
2. 建议使用现代浏览器（Chrome、Edge、Safari、Firefox）
3. 如果飞鸟背景没有显示，检查网络连接

## 🔧 自定义网站

你可以轻松修改：

1. **颜色主题**：编辑 `style.css` 中的颜色值
2. **游戏难度**：修改 `script.js` 中的 `GRID_SIZE` 和 `speed`
3. **背景效果**：调整 `script.js` 中 Vanta.js 的参数

## 📞 遇到问题？

- 确保所有文件（index.html、style.css、script.js）在同一个文件夹
- 确保网络连接正常（用于加载外部库）
- 使用浏览器的开发者工具（F12）查看错误信息
